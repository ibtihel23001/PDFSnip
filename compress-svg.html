FAQ

Common Questions

What does SVG minification do? +
SVG files are XML text. Minification removes whitespace, comments, metadata, and editor-specific data that design tools add. The visual output is identical — only file size changes.
Will my SVG still work after minification? +
Yes. The minified SVG renders exactly the same in browsers, design tools, and code. It's just more compact.
How much compression can I expect? +
SVGs exported from Figma, Illustrator, or Inkscape can be 30–70% larger than they need to be due to editor metadata and formatting. Minification typically saves 20–60%.
Is my SVG code uploaded anywhere? +
No. All processing is done in your browser using JavaScript. Your SVG files never leave your device.
