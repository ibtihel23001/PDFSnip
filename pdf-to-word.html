FAQ

Common Questions

How accurate is the conversion? +
For text-heavy PDFs (reports, articles, contracts), accuracy is very high — all text is extracted faithfully. Complex layouts like multi-column pages, forms, or text within images may not convert perfectly since the tool linearizes content into a standard Word paragraph flow.
Is my PDF uploaded to a server? +
No. Everything runs locally in your browser using PDF.js to extract text. Your file never leaves your device.
Why can't I convert scanned PDFs? +
Scanned PDFs contain images of text, not actual text characters. This tool extracts digital text from PDFs. Scanned documents require OCR (Optical Character Recognition) which is computationally intensive and not supported in this browser-based tool.
What is the output format? +
The output is a .docx file, compatible with Microsoft Word, Google Docs, LibreOffice, and all modern document editors. The file uses standard Open XML format.
