FAQ

Common Questions

Is my PDF uploaded to a server? +
No. All rotation is done using pdf-lib, a JavaScript library running entirely in your browser. Your file never leaves your device.
Can I rotate only specific pages? +
Yes! After loading your PDF, you'll see thumbnail previews of every page. Use the ↻ and ↺ buttons on each card to set individual rotations before saving.
Will my content and fonts be preserved? +
Absolutely. pdf-lib only changes the page rotation metadata — all content, fonts, images, and links are preserved exactly as-is.
What rotation angles are supported? +
You can rotate pages by 90°, 180°, or 270° (which is the same as -90°). These are the standard PDF rotation values supported by all PDF viewers.
