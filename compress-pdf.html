FAQ

Compress PDF — Common Questions

How much can you compress a PDF? +
It depends on the content. PDFs with many images can be compressed by 60–90%. PDFs with mostly text typically reduce by 10–30%. Our tool tries all available techniques to achieve maximum reduction.
Will compression affect text readability? +
Text in PDFs is vector-based and is not affected by compression. Only embedded images are resampled. At our default settings, text remains perfectly sharp and readable.
Is there a file size limit? +
We support PDF files up to 100MB. If your PDF is larger, consider splitting it first using our Split PDF tool, compressing each part, then merging them back.
Can I compress multiple PDFs at once? +
Yes! Drag and drop multiple PDF files at once, or select them all in the file picker. They'll all be compressed and available for download individually or as a ZIP.
